# NEONOVA — интернет-магазин на Flask

Готовый многостраничный интернет-магазин для запуска в **PyCharm** и деплоя на **Render**.

## Что есть в проекте

- главная страница
- каталог
- отдельные страницы товаров
- корзина
- страница о проекте
- 5 категорий товаров
- 15 товаров
- фильтрация
- сортировка
- локальная корзина через `localStorage`
- современный адаптивный дизайн
- имена создателей в проекте: **Александр Зейда, Игорь Бочаров, София Чукина**

## Как запустить локально

```bash
pip install -r requirements.txt
python app.py
```

Потом открой в браузере:

```bash
http://127.0.0.1:5000
```

## Как залить на Render

1. Загрузи проект в GitHub.
2. На Render создай **Web Service**.
3. Укажи:
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn app:app`
4. После деплоя сайт запустится автоматически.

## Где менять товары

Все товары лежат в файле `app.py` в списке `PRODUCTS`.

## Где менять картинки

Картинки лежат в папке:

```bash
static/images/products/
```

Можно просто заменить svg-файлы своими изображениями и оставить те же имена.

## Где менять логотип

Логотип лежит здесь:

```bash
static/images/logo.svg
```
