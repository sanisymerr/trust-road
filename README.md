# KidQuest Investor Landing

Готовый фронтенд-проект на **Vite + React**.

## Что внутри
- 1 сайт с красивым переключением между 3 лендингами:
  - для семей
  - для детей 7–10 лет
  - для подростков 11–14 лет
- блоки: hero, описание проекта, преимущества, фото и видео, FAQ, отзывы, footer CTA
- локальные изображения и локальный demo-video внутри проекта

## Локальный запуск
```bash
npm install
npm run dev
```

После этого открой адрес, который покажет Vite в терминале.

## Сборка production
```bash
npm run build
npm run preview
```

## Для Render
Это обычный статический Vite-проект.

### Build Command
```bash
npm install && npm run build
```

### Publish Directory
```bash
dist
```
